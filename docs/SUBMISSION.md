# OpenAI Build Week submission package

## Category

Education

## One-line pitch

MUSE turns GPT-5.6 teaching plans into a guide who physically walks, points and branches through a world according to what the learner actually notices.

## Project description

Most AI learning experiences stop at dialogue. MUSE makes reasoning spatial and inspectable. GPT-5.6 produces a strict lesson contract made only of known artwork, detail, gesture and effect IDs. A deterministic Three.js director then moves an articulated guide to the correct anchor, verifies her distance and facing, and only then opens the observation prompt. The learner's answer changes both the visible effect and the next physical destination.

The complete three-stop path works without credentials, while a live key activates GPT-5.6 planning and evidence-grounded Perspective Salon responses. Optional OpenAI Realtime voice, World Labs generation, shared rooms, local learner memory and mobile controls are isolated layers rather than dependencies of the judging path.

## What was built during Build Week

- Clean repository and fixed OpenAI-only service boundary.
- GPT-5.6 strict lesson and Salon contracts.
- Procedural articulated player, guide and Salon characters.
- Walk/turn/arrive/face/point/ask/listen/reflect behavior director.
- Scene correspondence assertions and evidence telemetry.
- Observation-driven physical route branching.
- Procedural Atlas plus optional audited Marble world.
- Protected Forge, Realtime relay, bounded rooms and local profiles.
- Desktop/mobile automated and visual QA.

The earlier MUSE project supplied research notes and disclosed assets only. Its mixed-agent runtime was not forked.

## Under-three-minute demo script

### 0:00-0:20: The premise

Open on the full museum world. Say: "A language model can describe a lesson. MUSE asks whether it can embody one."

Show the Evidence drawer briefly: GPT-5.6 state, manifest version and scene-sync metrics.

### 0:20-1:10: Embodied inquiry

Choose `How composition moves my attention`.

Let Mira walk to *Water Lilies*, turn and point. Emphasize that the prompt appears only after the anchor/facing check passes. Choose `A quiet surface` and show the ripple plus route to *The Bedroom*.

### 1:10-1:40: Branch proof

Restart or use a prepared second tab. Choose `Restless motion`; show that Mira goes to *La Grande Jatte* instead. State that GPT selects bounded semantic actions while deterministic code owns space and safety.

### 1:40-2:20: Learning map and layers

Complete the prepared path and show the three-evidence learning map. Open Salon to show the three embodied perspectives. Switch Atlas to Memory Garden. Briefly show that Forge and Voice report honest unavailable states without breaking text or movement.

### 2:20-2:45: Technical close

Show the contract and test result: one model boundary, no alternate LLM endpoint, no-key fallback, desktop/mobile canvas verification. End with: "MUSE makes an AI lesson something a learner can follow, test and remember in space."
